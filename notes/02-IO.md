# Basic Input / Output — Notes

## 8. Reading Input from User

### What is Input?

Input is data entered by the user for the program to process.

Examples:

* Number
* Name
* Marks
* Price

---

## Using `input()`

Python uses:

```python
input()
```

Example:

```python
name = input("Enter your name: ")
print(name)
```

Input:

```text
Sam
```

Output:

```text
Sam
```

---

## Reading Numbers

By default, `input()` reads data as a string.

```python
x = input("Enter number: ")
```

If user enters:

```text
10
```

Python stores:

```python
"10"
```

(string)

---

### Convert to Integer

```python
x = int(input("Enter number: "))
```

---

### Convert to Float

```python
price = float(input("Enter price: "))
```

---

## Reading Multiple Inputs

### Two inputs

```python
a = int(input())
b = int(input())
```

---

### In one line

```python
a,b = map(int,input().split())
```

Input:

```text
10 20
```

Output values:

```python
a=10
b=20
```

---

### Three numbers

```python
x,y,z = map(int,input().split())
```

---

## 10. Formatting Output

Formatting means displaying output in a required structure.

---

## Method 1 — Comma Style

```python
name="Sam"
age=15

print("Name:",name,"Age:",age)
```

---

## Method 2 — f-Strings (Best)

```python
name="Sam"
marks=92

print(f"{name} scored {marks}")
```

Output

```text
Sam scored 92
```

---

## Decimal Formatting

```python
pi=3.14159

print(f"{pi:.2f}")
```

Output

```text
3.14
```

---

## Aligning Output

```python
print(f"{'Name':10}{'Marks':5}")
```

---

## Example Table

```python
print("Name   Marks")
print("Raj    90")
```

---

## Common Formatting Needs

* Fixed decimal places
* Aligned columns
* Labels
* Units

Example:

```python
weight=60

print(f"Weight = {weight} kg")
```

## 11. Type Conversion

Type conversion means changing one data type into another.

---

## Types

* Integer (`int`)
* Float (`float`)
* String (`str`)
* Boolean (`bool`)

---

## Convert String to Integer

```python
x="10"

n=int(x)
```

---

## Convert Integer to Float

```python
x=5

y=float(x)
```

Result:

```text
5.0
```

---

## Convert Number to String

```python
n=50

s=str(n)
```

---

## Example

```python
a=input()
b=input()

print(int(a)+int(b))
```

---

## Why Conversion Needed?

Wrong:

```python
a="10"
b="20"

print(a+b)
```

Output:

```text
1020
```

String concatenation.

---

Correct:

```python
print(int(a)+int(b))
```

Output:

```text
30
```

---

## Implicit Conversion

Python sometimes converts automatically.

```python
x=5
y=2.5

print(x+y)
```

Output:

```text
7.5
```

---

## Explicit Conversion

Programmer converts manually.

```python
int()
float()
str()
```

## 12. Solving Simple Input-Output Problems

Basic approach:

### IPO Method

Input
Process
Output

---

## Problem 1

Read a number and print it.

Input:

```text
7
```

Output:

```text
7
```

Code:

```python
n=int(input())

print(n)
```

---

## Problem 2

Add two numbers.

Input

```text
5 6
```

Output

```text
11
```

Code

```python
a,b=map(int,input().split())

print(a+b)
```

---

## Problem 3

Find area of rectangle.

Formula:

```text
Area = length × width
```

```python
l=int(input())
w=int(input())

print(l*w)
```

---

## Problem 4

Convert Celsius to Fahrenheit

Formula:

F = \frac{9}{5}C + 32

```python
c=float(input())

f=(9/5)*c+32

print(f)
```

---

## Problem 5

Find average of 3 numbers

Average = \frac{a+b+c}{3}

```python
a,b,c=map(int,input().split())

print((a+b+c)/3)
```

# Common Beginner Errors

## Forgetting Conversion

Wrong

```python
a=input()
b=input()

print(a+b)
```

---

## Using Wrong Variable

```python
l=5
w=4

print(l+l)
```

Wrong formula.

---

## Incorrect Input Count

```python
a,b=map(int,input().split())
```

Needs exactly two values.
