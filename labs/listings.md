## Categories of Python Coding Questions (Questions Only)

### 1. Basic Input / Output

* Read a number and print it.
* Read a name and print “Hello, <name>”.
* Read two numbers and print their sum.
* Read a temperature and display it in Fahrenheit.
* Read length and width and print the area of a rectangle.

---

### 2. Arithmetic Problems

* Find the sum of three numbers.
* Find the average of five numbers.
* Find simple interest.
* Find compound interest.
* Find the area and perimeter of a circle.
* Convert seconds into hours, minutes, and seconds.

---

### 3. Conditional Statements (`if`, `if-else`)

* Check whether a number is positive or negative.
* Check whether a number is even or odd.
* Find the larger of two numbers.
* Find the largest of three numbers.
* Check whether a year is a leap year.
* Check whether a student passed or failed.

---

### 4. Nested Conditions / Decision Making

* Find grade based on marks.
* Check whether a person is eligible to vote.
* Determine ticket price based on age.
* Calculate electricity bill using slabs.
* Find income tax using tax brackets.

---

### 5. Looping (`for`, `while`)

* Print numbers from 1 to 100.
* Print even numbers from 1 to N.
* Find sum of first N natural numbers.
* Print multiplication table of a number.
* Find factorial of a number.
* Reverse counting from N to 1.

---

### 6. Pattern Problems

* Print a star triangle.
* Print an inverted triangle.
* Print Floyd’s triangle.
* Print number pyramid.
* Print hollow square pattern.

---

### 7. Number Theory Problems

* Check whether a number is prime.
* Find all factors of a number.
* Find GCD of two numbers.
* Find LCM of two numbers.
* Check whether a number is Armstrong.
* Check whether a number is a palindrome.
* Generate Fibonacci series.

---

### 8. String Problems

* Count vowels in a string.
* Reverse a string.
* Check whether a string is palindrome.
* Count words in a sentence.
* Convert lowercase letters to uppercase.
* Find frequency of each character.

---

### 9. List Problems

* Find largest element in a list.
* Find smallest element in a list.
* Find sum of all list elements.
* Remove duplicates from a list.
* Sort a list without built-in sort.
* Find second largest element.

---

### 10. Searching Problems

* Linear search for a number in a list.
* Binary search in a sorted list.
* Search for a character in a string.
* Count occurrences of a number.

---

### 11. Sorting Problems

* Implement Bubble Sort.
* Implement Selection Sort.
* Implement Insertion Sort.
* Sort student marks in ascending order.
* Sort names alphabetically.

---

### 12. Functions

* Write a function to add two numbers.
* Write a function to check prime.
* Write a function to find factorial.
* Write a function to reverse a string.
* Write a function to find maximum of three numbers.

---

### 13. Recursion (Intro)

* Factorial using recursion.
* Fibonacci using recursion.
* Sum of digits using recursion.
* Reverse a number using recursion.

---

### 14. Dictionaries

* Store student marks and display them.
* Find student with highest marks.
* Count frequency of words.
* Create a phone directory.
* Merge two dictionaries.

---

### 15. Problem-Solving / Logic Puzzles

* Find missing number in a sequence.
* Identify whether two numbers are amicable.
* Solve coin change problem.
* Find minimum notes for an amount.
* Simulate a basic ATM withdrawal.

---

### 16. Mini Real-World Programs

* Student grade calculator.
* Bank balance simulator.
* Simple billing system.
* Library fine calculator.
* Grocery bill generator.

---

### 17. File Handling Basics

* Write data into a file.
* Read and display file contents.
* Count number of lines in a file.
* Count words in a file.
* Copy contents from one file to another.

---

### 18. Beginner Debugging Challenges

* Find error in a given Python program.
* Predict output of a Python snippet.
* Fix a loop that runs infinitely.
* Correct indentation errors.

---

## Problem-Solving Skill Categories (Very Important)

These are the thinking skills behind the questions:

* Sequence problems
* Selection problems
* Iteration problems
* Pattern recognition problems
* Decomposition problems
* Simulation problems
* Greedy problems (basic)
* Brute force problems
* Algorithmic thinking problems

---

A good progression for high schoolers is:

1. Input/Output
2. Arithmetic
3. Conditions
4. Loops
5. Patterns
6. Number Theory
7. Strings
8. Lists
9. Searching/Sorting
10. Functions/Recursion
11. Mini Projects
